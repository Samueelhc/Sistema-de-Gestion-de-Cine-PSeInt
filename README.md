# Sistema de Gestion de Cine en PSeInt

Sistema interactivo desarrollado en PSeInt que simula la gestión completa de un complejo de cines, permitiendo seleccion de peliculas, venta de entradas, selección de asientos y compras en dulcería.

---

## 🚀 Funcionalidades

- Cartelera interactiva con visualización de películas, salas y horarios  
- Sistema de reserva y cancelación de asientos  
- Módulo de dulcería con selección de productos (comida, bebidas, alcohol, dulces) 
- Procesamiento y diferentes formas de pago.  
- Generación de facturación detallada  
- Validación de datos para garantizar estabilidad del sistema  

---

## 🛠️ Características Técnicas

- Programación estructurada en PSeInt  
- Uso de matrices para la gestión de salas  
- Modularización mediante subprocesos  
- Validación de entradas del usuario  
- Simulación de procesos administrativos  

---

## 🎯 Objetivo

Simular el funcionamiento operativo de un sistema de cine aplicando fundamentos de programación estructurada y lógica algorítmica.
